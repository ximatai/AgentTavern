## 风险与待关注项

### 1. 本地 Agent 接入差异

不同本地 Agent 的输入输出协议可能差异较大，需要 adapter 层充分隔离。

### 2. 流式输出标准化

不同 Agent 的流式粒度可能不一致，需要统一成稳定事件格式。

### 3. Mention 匹配歧义

若房间里存在重名成员，需要补充唯一标识或命名约束。

### 4. 审批时效与状态同步

审批请求需要处理超时、owner 离线、重复点击、会话取消等问题。
